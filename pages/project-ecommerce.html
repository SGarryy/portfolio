Project 03 — Sales Intelligence

E-commerce Sales Analytics

Deep-dive EDA on a multi-table e-commerce dataset — orders, customers, products, and returns. Uncovered revenue trends, top products, customer segments, and the financial impact of returns. Built both a Jupyter Notebook analysis and a PyQt5 desktop dashboard for interactive visualization.

Includes moving average forecasting, repeat buyer segmentation, and business-ready chart outputs.

What Was Built
📓
Jupyter Notebook Analysis

Full EDA covering product revenue, repeat buyers, monthly trends, return analysis, and moving average forecasting — all in a single reproducible notebook.

🖥️
PyQt5 Desktop Dashboard

Interactive GUI with analysis buttons for monthly revenue trend, top 5 products, customer segments, returns impact summary, and forecast — styled with Qt Stylesheets.

📦
Returns Impact Analysis

Quantified revenue lost to returns by category and month — surfacing which products had the highest return rates and their effect on net revenue.

📈
Moving Average Forecasting

3-month moving average applied to revenue data to smooth seasonality and project near-term sales trends — available in both the notebook and the dashboard.
