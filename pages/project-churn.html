Project 02 — Classification & Deployment

Customer Churn Prediction

End-to-end ML pipeline predicting customer churn on ~7,000 Telco records with 20+ features. Built a Random Forest classifier inside a full sklearn pipeline, handled class imbalance with SMOTE, and deployed a real-time prediction app on Streamlit Community Cloud.

Users can input customer details and receive an instant churn risk verdict — High or Low — with a probability score.

ML Pipeline
01
Data Cleaning & Preprocessing

Handled missing values in TotalCharges, encoded categorical features with One-Hot Encoding, and scaled numerical features with StandardScaler — all inside a single sklearn Pipeline.

02
Class Imbalance — SMOTE Oversampling

The Telco dataset has ~73% "Stay" vs ~27% "Churn". Applied SMOTE to oversample the minority class — preventing the model from simply predicting "Stay" for everyone.

03
Random Forest Classifier Training

Trained a Random Forest inside the pipeline. Evaluated with accuracy, precision, recall, F1, and ROC-AUC (≈ 0.81). Contract type emerged as the strongest churn predictor.

04
Model Serialization via Joblib

Saved the trained pipeline as model.pkl — the Streamlit app loads this directly for real-time predictions without re-training.

05
Streamlit Deployment

Deployed on Streamlit Community Cloud. Users fill in customer details and instantly receive a churn risk label (High/Low) with a probability score.
